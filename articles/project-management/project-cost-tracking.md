## <a name="labor-cost-tracking-view"></a><span data-ttu-id="aa029-108">Приказ праћења цене рада</span><span class="sxs-lookup"><span data-stu-id="aa029-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="aa029-109">На страници **Пројекти**, на картици **Праћење** можете изабрати **Праћење** > **Цена** да бисте отворили приказ **Праћење трошкова** и видели напредак утрошка рада за сваки задатак у плану пројекта.</span><span class="sxs-lookup"><span data-stu-id="aa029-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="aa029-110">Овај приказ такође упоређује стварну цену рада утрошену на задатку са планираном ценом рада.</span><span class="sxs-lookup"><span data-stu-id="aa029-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="aa029-111">Project Operations користи следеће формуле за израчунавање метрика цене рада:</span><span class="sxs-lookup"><span data-stu-id="aa029-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="aa029-112">**Планирана цена**: Процењена цена свих додела ресурса на сваком задатку чвора листа</span><span class="sxs-lookup"><span data-stu-id="aa029-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>

- <span data-ttu-id="aa029-115">**Преостала цена**: Процена цене на завршетку – Стварна цена</span><span class="sxs-lookup"><span data-stu-id="aa029-115">**Remaining Cost**: Cost estimate at complete  – Actual cost</span></span>

- <span data-ttu-id="aa029-117">**Одступање од цене**: Планирана цена – Процена цене на завршетку</span><span class="sxs-lookup"><span data-stu-id="aa029-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>
