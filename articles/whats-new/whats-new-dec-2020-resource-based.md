# <a name="whats-new-december-2020---project-operations-for-resourcenon-stocked-based-scenarios"></a>Шта је ново децембра 2020. – Project Operations за сценарије засноване на ресурсима/без залиха

Ова тема се односи на следеће компоненте и верзије услуге Dynamics 365 Project Operations:

- Project Operations у Dataverse окружењу верзије 4.5.0.134

За информације о начину ажурирања на ово издање погледајте чланак [Ажурирање услуге Project Operations у Finance окружењу](ur5-nonstocked-installation.md).
